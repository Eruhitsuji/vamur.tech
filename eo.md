# VAMURTECH Esperanta Traduko

## Latina kanono

> **Vita brevis; ars longa, memoria servata usuque renovato; amor per artes technicas longas omnia vincit.**

## Esperanta traduko

> **La vivo estas mallonga; arto kaj tekniko estas longaj, kun memoro konservata kaj uzo renovigata; amo, per longaj teknikaj artoj, venkas ĉion.**

## Plivastigita interpreto

La homa vivo estas mallonga, kaj tio, kion unu homo povas rekte fari, estas limigita.  
Tamen lernado, metio, scio, verkoj, registroj kaj datumoj povas daŭri trans unu homo kaj trans unu generacio.  
Ili ne restas vivantaj simple pro senmova konservado.  
Ili daŭras ĉar memoro estas konservata dum uzo, metodo, rimedo kaj praktiko estas renovigataj.

En ĉi tiu devizo, **ars** ne signifas nur belarton.  
Ĝi inkluzivas metion, lertecon, metodon, scion, inĝenierion, komputadon, komunikadon, dokumentadon kaj ĉiun disciplinan manieron per kiu homoj komprenas naturon, faras ilojn, ordigas aferojn, konservas informojn kaj transdonas scion al aliaj.

**Memoro estas konservata**, ĉar scio, verkoj kaj datumoj povas esti registritaj, kopiitaj, protektitaj kaj transdonitaj.  
Unu korpo aŭ unu aparato povas perei, sed tio, kio estas skribita, duobligita, rezervkopiita kaj transdonita, povas resti trans unu homo kaj trans unu epoko.

**Uzo estas renovigata**, ĉar iloj, medioj, operaciaj metodoj kaj teknikaj paradigmoj ŝanĝiĝas tra la tempo.  
La sama memoro povas esti konservata per novaj medioj; la sama scio povas esti uzata laŭ novaj manieroj.  
Tio, kio iam okazis per papero, tabuloj aŭ simplaj instrumentoj, poste povas okazi per komputiloj, retoj kaj aliaj rimedoj.  
Renovigo ne detruas memoron; ĝi vivigas memoron.

Tiel arto kaj tekniko estas longaj ne ĉar ili neniam ŝanĝiĝas, sed ĉar ili daŭras per konservata memoro kaj renovigata uzo.  
Tekniko, kiu ne estas transdonata, malaperas.  
Tekniko, kiu neniam estas renovigata, rigidiĝas.  
Sed tekniko, kiu konservas memoron kaj renovigas uzon, kreskas tra la tempo.

Fine, amo estas la principo, kiu direktas teknikajn artojn.  
Sen amo, tekniko povas fariĝi nura potenco.  
Kiam amo gvidas teknikon, scio kaj iloj turniĝas al protektado, helpado, kulturado kaj riĉigo de vivo.  
Ĉi tie, venki ne signifas regi.  
Ĝi signifas superi forgeson, fragilecon, malfacilaĵojn kaj interrompon per amo kaj arto.
