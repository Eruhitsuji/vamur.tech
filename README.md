# VAMURTECH

This repository contains the public GitHub Pages site for **VAMURTECH**.

Canonical motto:

> **Vita brevis; ars longa, memoria servata usuque renovato; amor per artes technicas longas omnia vincit.**

Custom domain:

> **vamur.tech**

## Files

- `index.md` — English-centered top page
- `motto.md` — Latin canon
- `ja.md` — Japanese translation and commentary
- `en.md` — English translation and commentary
- `eo.md` — Esperanto translation and commentary
- `NOTICE.md` — official canon and attribution notice
- `LICENSE.md` — MIT License with official canon notice
- `assets/logo.png` — transparent logo image
- `assets/logo_original.png` — original generated logo image
- `CNAME` — GitHub Pages custom domain setting
- `_config.yml` — GitHub Pages / Jekyll configuration

## GitHub Pages

Recommended setting:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`
- Custom domain: `vamur.tech`
